## Valuation Metrics

The application analyzes stocks using the following metrics:

- **EV/EBITDA** - Enterprise value to earnings before interest, taxes, depreciation
- **P/E Ratio** - Price-to-earnings multiple
- **P/B Ratio** - Price-to-book value
- **EV/Revenue** - Enterprise value to revenue
- **FCF Yield** - Free cash flow yield
- **Debt-to-Equity** - Leverage analysis
- **Cash Flow Multiple** - Operating cash flow valuation
